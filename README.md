# Rust_add_calender_event
## モチベーション
Googleカレンダーにイベントを追加するCLIをRustでつくりたい。
RustでCLIを作ることでRustの学習をしつつ自分の普段のライフスタイルに即したCLIを作ることで継続的な改善をしてテストなどの練習をしたい。
